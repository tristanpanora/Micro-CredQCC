What exactly is a favicon?

A favicon is a 16x16 pixel icon, and its short for "favorite icon". It is commonly used as a way of putting a recognizable brand logo on a website. These little icons appear everywhere on the internet, but a common place to see a favicon is in the little image next on the tab at the top of a web browser. They are very small, only 16x16 pixels, so they are best used when displayed in a very small setting.

The little icons in the tabs are favicons! Click any favicons below to go to the respective site.

How are favicons created?

Favicons can be created by anyone! There are many programs that exist in order to design and create favicons. You can use a program to draw them by selecting color and filling in pixels, or you can use a program to code a favicon with binary or hexadecimal. Big brands such as Facebook or Twitter use shrunken down versions of their company logos usually. Typically a favicon is 16x16 pixels, but it can be 32x32 and use a wide range of colors.

How did I create the favicon for this site?

I used Code.org (click link to create your own) to create my favicon. This link will lead directly to the color pixelation freeplay, but if you want an in depth explaination on how to use binary and hexadecimal to create your favicon, there are videos on the site to help you. Essentially, I use binary to represent colors in each pixel. Starting with 12 bits per pixel is good, because it allows for a wide range of colors.
